  한글일본어입력기 (HangulJapaneseIME) for Windows
  한글 자판으로 일본어를 입력하는 키보드 도구
═══════════════════════════════════════════════════════════

  ■ 요구 사항
    - Windows 10 / 11 (64-bit)

  ■ 설치 방법
    1. install.bat 더블클릭
    2. 완료! (자동으로 시작 프로그램에 등록됩니다)

  ■ 사용 방법
    Ctrl+Space    입력기 켜기/끄기 (토글)

    켜진 상태에서 한글 두벌식 자판 배열로 타이핑하면
    히라가나로 변환되어 입력됩니다.

    키보드 매핑 (두벌식 기준):
      d→ㅇ k→ㅏ  →  あ (아)
      f→ㄹ l→ㅣ  →  り (리)
      r→ㄱ k→ㅏ  →  か/が (가)
      x→ㅌ h→ㅗ  →  と (토)

    입력 예시:
      dkflrkxh          (아리가토)    → ありがと
      dkflrkxhndhdwkdlaktn (아리가토우고자이마스) → ありがとうございます

    조작:
      스페이스 / 엔터  → 변환 확정
      백스페이스       → 한 글자 삭제
      Esc             → 입력 취소
      Ctrl+Space      → 입력기 켜기/끄기

  ■ 제거 방법
    uninstall.bat 더블클릭
